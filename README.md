# Oman Elite | Luxury Tourism Web Application

Oman Elite is a high-end luxury tourism web application designed to facilitate premium travel experiences, such as private jet and yacht bookings, within Oman.

## Project Overview
This project was developed as an academic assignment to demonstrate full-stack development capabilities and "quiet luxury" brand integration.

## Tech Stack
* **Frontend:** HTML5, CSS3, JavaScript (Responsive Design). 
* **Backend:** PHP. 
* **Database:** MySQL for secure booking and inventory management.

## Key Features
* **Luxury UI:** Designed with a "quiet luxury" aesthetic to appeal to high-net-worth individuals. 
* **Booking Management:** Robust backend logic for handling travel reservations.
* **Database Integration:** Fully functional relational database for managing users and bookings.

## 📁 Repository Structure
* `/css`: Custom styling for the luxury aesthetic.
* `/js`: Frontend interactivity.
* `index.php`: Main landing page and booking logic.
* `oman_elite.sql`: SQL export for the booking system.
